# MonOpSy

A simple operating system project.

## Overview
MonOpSy is a lightweight OS implementation designed for educational purposes.

## Features
- Basic kernel architecture
- Process management
- Memory management
- Simple I/O operations

## Getting Started

### Prerequisites
- C/C++ compiler
- Build tools (Make, CMake, etc.)

### Building
```bash
make build
```

### Running
```bash
make run
```

## Project Structure
```
MonOpSy/
├── kernel/
├── drivers/
├── boot/
└── docs/
```
